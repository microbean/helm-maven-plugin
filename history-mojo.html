helm:history

Full name:

org.microbean:helm-maven-plugin:2.7.2.1.0.1:history

Description:

Retrieves the history of a release.

Attributes:

  • Requires a Maven project to be executed.

Required Parameters

Name Type Since Description
releaseName String - The name of the release.

Optional Parameters

Name Type Since Description
clientConfiguration Config - The Config describing how a DefaultKubernetesClient should connect to a Kubernetes cluster.
max int - The maximum number of versions to return.
releaseHistoryListeners List - A List of ReleaseHistoryListeners whose elements will be notified of each item in the history.
Alias is: releaseHistoryListenersList.
skip boolean - Whether to skip execution.
Default value is: false.

Parameter Details

clientConfiguration:

The Config describing how a DefaultKubernetesClient should connect to a Kubernetes cluster.
  • Type: io.fabric8.kubernetes.client.Config
  • Required: No

max:

The maximum number of versions to return.
  • Type: int
  • Required: No

releaseHistoryListeners:

A List of ReleaseHistoryListeners whose elements will be notified of each item in the history.
  • Type: java.util.List
  • Required: No
  • Alias: releaseHistoryListenersList

releaseName:

The name of the release.
  • Type: java.lang.String
  • Required: Yes

skip:

Whether to skip execution.
  • Type: boolean
  • Required: No
  • Default: false

# helm-maven-plugin

A [Maven plugin][maven-plugin] for working with [microbean-helm][], a Java version of [Helm, the Kubernetes package manager][helm].

## Prerequisites

* [Maven 3.5.0][maven]

## Documentation

The project is [fully documented][documentation].
